Logo

Der Name des Projektes vereint bereits die beiden wichtigsten Elemente unseres Konzepts, eine Pinnadel und Information. Unser Gedanke beim Entwurf des Logos war es, dies so präzise wie möglich in ein Symbol wiederzuspiegeln. Der Buchstabe "P" bot sich sehr gut dazu an. Das jetzige Logo zeigt in rot den Buchstaben, erinnert in seiner Gesamtheit an eine Pinnadel und durch die Einarbeitung eines Ausrufezeichens werden die Begriffe "Dringlichkeit" und "Information" betont.

CSS Methodology

  1. Code style rules

    • Einrückungen: 1 Tab (4 Leerzeichen)
    • Einheiten: px, em, percentage
    • Ordnerstruktur: Überordner "src", Inhalt:

      • `--css
|  `--partials
`--fonts
`--images
`--js

  2. Pre-Processor: SASS

  3. Class naming conventions (Bootstrap)

Component Libraries

  • Bootstrap

    1. Buttons

      Buttons werden zum Bestätigen der Auswahl des Studiengangs verwendet oder zum Download von Notizen.



      <button type="button" class="btn btn-default"><i class="material-icons">file_download</i></button>
<button type="button" class="btn btn-default">OK</button>

    2. Dropdown

      Das Dropdown-Feld wird für die Startseite zum Auswählen des gewünschten Studiengangs eingesetzt.

      • Action
      • Another action
      • Something else here
      • Separated link

      <div class="dropdown">
    <button class="btn btn-default dropdown-toggle" type="button" id="dropdownMenu1" data-toggle="dropdown" aria-haspopup="true" aria-expanded="true">
        Dropdown
        <span class="caret"></span>
    </button>
    <ul class="dropdown-menu" aria-labelledby="dropdownMenu1">
        <li><a href="#">Action</a></li>
        <li><a href="#">Another action</a></li>
        <li><a href="#">Something else here</a></li>
        <li role="separator" class="divider"></li>
        <li><a href="#">Separated link</a></li>
    </ul>
</div>

    3. Navigation

      Der Navigationsbutton "Zurück" wird auf der Seite mit der Pinnwand angezeigt und dient zum zurücknavigieren auf die Startseite.



      <button type="button" class="btn btn-default"><i class="material-icons">backspace</i></button> <button type="button" class="btn btn-default"><i class="material-icons">close</i></button>

  • Material Icons

    1. backspace Backspace
      Wird verwendet, um von der Pinnwand wieder auf die Studiengangsauswahlseite zu gelangen.
    2. close Close
      Erscheint in der Detailansicht, bei Klick darauf schließt sich diese Ansicht.
    3. file_download Download
      In der Detailansicht gibt es auch die Möglichkeit, sich das Dokument zu downloaden.
